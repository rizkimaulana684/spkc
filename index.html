Metodologi Proyek

Penjelasan pendekatan model-driven dan data-driven

Model-Driven: AHP + TOPSIS

AHP (Analytic Hierarchy Process) digunakan untuk menghitung bobot 5 kriteria keputusan melalui matriks perbandingan berpasangan skala Saaty (1-9). Hasil: Avg User Rating (41.6%), Rating Google (26.2%), Total Reviews (16.1%), Price (9.9%), Time (6.2%). Uji konsistensi menghasilkan CR = 0.0153 (< 0.10), sehingga bobot dinyatakan valid.

TOPSIS (Technique for Order Preference by Similarity to Ideal Solution) melakukan perangkingan 20 alternatif destinasi berdasarkan jarak terhadap solusi ideal positif (A+) dan negatif (A-). Closeness Coefficient (CC) tertinggi menunjukkan destinasi terbaik.

Data-Driven: Random Forest Classifier

Random Forest (100 trees, max_depth=10) dilatih pada 10.000 data rating user untuk memprediksi kelas kepuasan wisatawan (Tinggi/Sedang/Rendah). Fitur: Rating, Price, Time_Minutes, Category, City, Age.

Accuracy: 38.9% — rendah karena rating user sangat subjektif dan fitur terbatas. Namun model tetap bermanfaat untuk probabilitas prediksi per kelas dan feature importance analysis. Feature terpenting: Age (41.9%), Price (15.9%), Rating (14.7%).

Hybrid Integration

Kedua pendekatan divalidasi silang menggunakan Spearman Rank Correlation (ρ = 0.669, p = 0.001), menunjukkan korelasi kuat. 11 dari 20 destinasi memiliki ranking konsisten (selisih ≤ 3). Taman Sungai Mudal berada di peringkat 1 pada kedua metode, mengkonfirmasi keandalan hasil.

Pada fitur Prediksi, sistem menggabungkan kedua metode: TOPSIS menghitung ranking berdasarkan bobot kriteria user, sementara Random Forest memprediksi probabilitas kepuasan. Skor akhir = 60% TOPSIS + 40% RF untuk rekomendasi yang lebih komprehensif.

Dataset

Sumber: Indonesia Tourism Destination (Kaggle — aprabowo)
tourism_with_id.csv: 437 destinasi di 5 kota (Jakarta, Yogyakarta, Bandung, Semarang, Surabaya), 6 kategori
tourism_rating.csv: 10.000 rating dari 300 user (skala 1-5)
user.csv: 300 profil user (lokasi, usia 18-40 tahun)
Preprocessing: Missing value handling (median imputation), encoding kategori & kota, aggregasi rating per destinasi, normalisasi
